# Fortémi — March 2026

*Fortémi is the agent-memory server. It keeps notes for an AI agent and lets the agent search them by meaning, not just by exact words. You run it yourself, on your own machine, with one Docker command. This is the full server. HotM is a desktop app that uses it. fortemi-react is the same server built to run in a web browser.*

## TL;DR

March was a quiet month. No new version shipped. The last release was in late February, and the next one landed in early April. In between, the work was light and behind the scenes — steady groundwork, not new features you can use yet. There is nothing new to install this month. If you are running Fortémi, you can keep running what you have. This is the honest short version: a calm, between-releases month.
