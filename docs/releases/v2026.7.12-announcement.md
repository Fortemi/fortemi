# v2026.7.12 Release Announcement

Fortemi v2026.7.12 is the corrective release for the v2026.7.11 publication.
It preserves the same Knowledge Shard formats, profiles, compatibility window,
and nine-cell conformance evidence.

## Fixed

- The bundled documentation shard and its receipt are regenerated against the
  matching v2026.7.12 API instead of carrying a stale server-version marker.
- Third-party dependency verification ignores generated `target/` and
  `node_modules/` trees, so persistent runner caches cannot create false policy
  failures from dependency-owned Dockerfiles.

## Install

```bash
docker pull ghcr.io/fortemi/fortemi:2026.7.12
docker pull ghcr.io/fortemi/fortemi:bundle-2026.7.12
```

Production deployment records should retain the immutable digests resolved
from these versioned tags.
