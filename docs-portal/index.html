Fortémi Docs

Documentation for Fortémi server and browser packages

Fortémi is an AI-enhanced knowledge platform with hybrid semantic search, W3C SKOS tagging, graph exploration, multimodal ingestion, and agent integrations. This page routes readers and crawlers to the two published docsites.

Fortémi Server

The self-hosted Fortémi server: Rust API, PostgreSQL with pgvector and PostGIS, hybrid search, MCP server, multi-provider LLM inference, multimodal ingestion, encryption, and operations guidance.

  • Deployment, backup, release, and troubleshooting runbooks
  • API, MCP, real-time events, auth, and permissions reference
  • Bot-friendly content and discovery provided by the docsite build
